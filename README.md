# TrainCrewMoniter

TrainCrewMoniterは、溝月レイル/Acty様製作の列車運転ゲーム「TRAIN CREW」で動作する、TASC機能付きモニタリングソフトです。

# 使い方
1、TRAIN CREWの操作設定→外部デバイス入出力を「有効」に設定してください。

2、Zipファイルを解凍後、TrainCrewMoniter.exeをダブルクリックして起動してください。
